## how to install

### .env file

```
DATABASE=./data
```

## how to use

### banks

```
bash ./src/banks.sh 
```

### bank

```
bash ./src/bank.sh 9
```

### cnpj

```
bash ./src/cnpj.sh 99999999999999
```

### ddd

```
bash ./src/ddd.sh 99
```

### participants of pix

```
bash ./src/pix.participants.sh 
```

### taxas

```
bash ./src/taxas.sh 
```

### feriados

```
bash ./src/feriados.sh 2023
```

### cep

```
bash ./src/cep.sh 99999999
```

### isbn

```
bash ./src/isbn.sh 9999999999999
```
